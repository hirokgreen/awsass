Automate "Web Application Submission" System

To build a platform to help the students & teachers in the context of submitting projects.

About Us

This project is to develop a system that will help to submit project to it’s teachers. To build a platform to help the students & teachers in the context of submitting projects. This project is aimed to enable the student to submit their assignment or project online without submitting any physical file; The proposed system decreases the complexity of managing projects for student by providing them with the current status of their projects and the progresses with their supervisors. The main goal of this project is to design and implement online assignment submission and provide an interface use for uploading test program by instructors, who would be able to evaluate assignments automatically. The interface use to invoke different testing program by teachers, So Save the time and cost for teachers by enabling them to put up a fast response for students as well as increasing the quality of the feedback provided to students.
Nowadays, higher education has become one of the most necessary factors in daily life. In modern academic curriculum, the projects are becoming more and more important. Time limitation is an undeniable issue that people face in daily life and work. Student projects are not an exception and most of student and supervisors face this issue during the project progress. Students hardly can manage their time to do their projects and meet their supervisors. This paper will discuss on prototype that will attempt to help in overcoming these issues and provide a good experience for the users. Upload and set assignment datelines with ability to review and edit assignments for grading purposes directly in the browser. Students able to view on assignment dateline alerts and submit assignment online and see if the assignments submitted have been graded. Consistent grading in parallel with the review process while giving feedback on the students’ performance.
